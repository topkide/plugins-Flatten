# Flatten (평탄화)

플레이어가 **지나가는 곳**을 지정한 **기준 Y 좌표**를 기준으로 평탄화하는 독립 플러그인.
MySQL/Redis/Core 의존 없음. Paper 1.21.8.

## 동작 규칙

- 기준 Y **이상(포함)** 의 모든 블록 → **제거(공기)**
- 기준 Y **바로 아래(Y-1)** → **잔디 블록**으로 채움 (빈 공간 매움)
- 결과: 기준 Y 높이에 평평한 잔디 표면이 형성됨
- 명령어로 **ON** 시, 플레이어가 이동해 새 블록으로 들어갈 때마다 주변 반경을 자동 평탄화

## 명령어 (`/평탄화`, alias `/flatten`)

| 명령어 | 설명 |
|---|---|
| `/평탄화 on [Y] [범위]` | 켜기. Y·범위 인라인 지정 가능 |
| `/평탄화 off` | 끄기 |
| `/평탄화 y <값>` | 기준 Y 좌표 지정 |
| `/평탄화 범위 <값>` | 반경(칸) 지정 |
| `/평탄화 정보` | 현재 설정 확인 |
| `/평탄화 리로드` | config.yml 재적용 (`flatten.admin`) |

권한: `flatten.use` (기본 op), `flatten.admin` (기본 op)

## 예시

```
/평탄화 y 64        # 기준 높이 Y=64
/평탄화 범위 5       # 반경 5칸 (11x11)
/평탄화 on          # 켜고 걸어다니면 지나간 자리가 평탄화됨
/평탄화 off
```

또는 한 줄로:
```
/평탄화 on 64 5     # Y=64, 반경 5칸으로 즉시 켜기
```

## config.yml

```yaml
default-y: 64          # 기본 기준 Y
default-radius: 3      # 기본 반경
max-radius: 20         # 지정 가능한 최대 반경 (성능 보호)
surface-block: GRASS_BLOCK   # Y-1 채움 블록
circle: false          # true면 원형 반경
apply-physics: false   # 블록 변경 시 물리 효과 (false 권장)
disabled-worlds: []    # 평탄화 비활성 월드
```

## 빌드

```bash
./gradlew build
# 결과물: build/libs/Flatten-1.0.0.jar
```

## 참고 (성능/주의)

- 상태는 메모리 전용 → 접속 종료/서버 재시작 시 초기화됩니다.
- 이동 시 **블록이 바뀔 때만** 처리하며, 지형 표면 높이까지만 스캔하고
  로드된 청크만 처리하여 렉을 최소화했습니다.
- `apply-physics: false` 로 모래 낙하·물 흐름 등 연쇄 갱신을 방지합니다.
- 되돌리기(undo) 기능은 없습니다. 넓은 반경 사용 전 백업을 권장합니다.
